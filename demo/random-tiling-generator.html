About Isohedral Tilings

This demo showcases isohedral tilings - periodic patterns where all tiles are congruent and have the same symmetries. Each click generates a new random combination of:

  • Tiling Type: One of 81 different isohedral tiling types
  • Parameters: Geometric parameters that control the tile shape
  • Edge Shapes: Custom curves for tile edges (I=straight, J/S/U=curved)
  • Colors: Random color scheme for the pattern
  • Transform: Position, rotation, and scale of the pattern
